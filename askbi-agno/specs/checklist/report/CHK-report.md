## 检查项详情

### CHK-report-人事-001 人事考勤报表生成

**关联需求**: REQ-report-生成报表
**目的**: 验证人事考勤报表生成器正确工作
**方法**: 上传测试数据文件，生成报表
**等级**: 阻塞

**检查步骤**:
1. 上传明细表与汇总表
2. 选择人事考勤报表类型
3. 验证报表文件生成
4. 验证统计信息正确 (行数、列数、标黄、问题)
5. 验证预览数据正确

**预期结果**:
- 报表文件正确生成
- 统计信息准确

### CHK-report-脱敏-006 数据脱敏功能

**关联需求**: REQ-report-数据脱敏
**目的**: 验证脱敏功能正确性
**方法**: 脱敏配置与验证
**等级**: 重要

**检查步骤**:
1. 打开报表脱敏预览
2. 验证自动检测敏感列
3. 配置脱敏方法并应用
4. 验证脱敏文件生成
5. 关闭脱敏，验证脱敏文件删除
6. 原始文件不受影响

**预期结果**:
- 脱敏正确应用
- 原始文件安全
- 切换功能正常

### CHK-report-管理-005 报表管理功能

**关联需求**: REQ-report-报表管理
**目的**: 验证报表列表/预览/下载/删除/重命名
**方法**: CRUD 端到端测试
**等级**: 阻塞

**检查步骤**:
1. 查看报表列表
2. 预览报表数据
3. 下载报表文件 (原始与脱敏)
4. 重命名报表
5. 删除报表，验证文件与记录清理

**预期结果**:
- 所有管理操作正常
- 用户数据正确隔离

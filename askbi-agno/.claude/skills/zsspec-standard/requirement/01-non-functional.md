# 非功能需求

**版本**: v0.1

---

## NFR-性能-响应时间

| 场景 | 要求 |
|------|------|
| 页面首次加载 | < 3秒 |
| 常规API响应 | < 500毫秒 |
| 复杂查询 | < 2秒 |

## NFR-性能-并发

| 指标 | 要求 |
|------|------|
| 并发用户数 | 支持 100+ |
| 单表数据量 | 支持百万级 |

## NFR-安全-认证

| 项目 | 要求 |
|------|------|
| 认证方式 | JWT Token |
| Token有效期 | Access 2小时，Refresh 7天 |
| 权限模型 | RBAC |

## NFR-安全-加密

| 项目 | 要求 |
|------|------|
| 密码存储 | BCrypt |
| 传输加密 | HTTPS，TLS 1.2+ |
| 敏感信息 | 脱敏展示 |

## NFR-可用-兼容

| 项目 | 要求 |
|------|------|
| 浏览器 | Chrome/Firefox/Edge/Safari 最新2版本 |
| 分辨率 | ≥ 1366x768 |

## NFR-维护-规范

| 项目 | 要求 |
|------|------|
| 代码规范 | ESLint/Prettier通过 |
| 单元测试 | 核心业务覆盖率 > 80% |

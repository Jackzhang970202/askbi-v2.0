# API契约

**版本**: v0.1

---

## 基础规范

| 项目 | 说明 |
|------|------|
| 基础路径 | /api/v1 |
| 认证 | Bearer Token (JWT) |
| 格式 | JSON |

## 请求方法

| 方法 | 用途 |
|------|------|
| GET | 查询 |
| POST | 创建 |
| PUT | 更新 |
| DELETE | 删除 |

## 响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

## HTTP状态码

| 码 | 说明 |
|----|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未登录 |
| 403 | 无权限 |
| 404 | 不存在 |
| 500 | 服务器错误 |

## 分页格式

```json
{
  "list": [],
  "total": 100,
  "pageNum": 1,
  "pageSize": 10
}
```

## 权限格式

| 操作 | 格式 | 示例 |
|------|------|------|
| 查询 | {模块}:{资源}:list | user:account:list |
| 新增 | {模块}:{资源}:add | user:account:add |
| 修改 | {模块}:{资源}:edit | user:account:edit |
| 删除 | {模块}:{资源}:remove | user:account:remove |

## 通用接口

| 接口 | 方法 | 路径 |
|------|------|------|
| 分页查询 | GET | /{模块}/{资源}/list |
| 查询详情 | GET | /{模块}/{资源}/{id} |
| 创建 | POST | /{模块}/{资源} |
| 更新 | PUT | /{模块}/{资源}/{id} |
| 删除 | DELETE | /{模块}/{资源}/{id} |

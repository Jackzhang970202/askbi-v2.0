## 页面设计

### 智能体管理页

#### 页面结构
- **顶部栏**: 标题"智能体管理" + "新建智能体"按钮
- **主区域**: 卡片网格布局（每行 3 张卡片）
  - 每张卡片展示智能体摘要信息
  - 点击卡片打开编辑面板（右侧抽屉或模态框）
- **底部**: 对话测试面板（从卡片底部展开或独立模态框）

#### 卡片结构
- 显示名称（display_name）
- 名称（name，灰色小字）
- 描述（description，截断两行）
- 状态标签：启用（绿色）/ 禁用（灰色）
- 内置标签：内置智能体显示"内置"badge
- 绑定技能数量徽标：如"技能 x3"
- 操作按钮：编辑、测试、删除（内置智能体不显示删除）

#### 交互流程
1. 页面加载 → 调用 `GET /agents` 获取列表 → 渲染卡片网格
2. 点击卡片"编辑" → 右侧展示编辑面板
3. 编辑后点击"保存" → 调用 `PUT /agents/{id}` → 刷新卡片
4. 点击"新建智能体" → 编辑面板切换为创建表单
5. 点击"保存" → 调用 `POST /agents` → 新增卡片
6. 点击"删除" → 确认弹窗 → 调用 `DELETE /agents/{id}` → 移除卡片
7. 点击"测试" → 展开对话测试面板
8. 编辑面板中点击"绑定技能" → 展示技能多选列表 → 调用 `POST /agents/{id}/bind-skills`

#### 组件

- **卡片网格**
  - 响应式布局：大屏 3 列、中屏 2 列、小屏 1 列
  - 卡片悬停高亮效果
  - 内置智能体卡片左上角"内置"badge
  - 右上角状态指示点（绿色=启用，灰色=禁用）

- **编辑面板（模态框/抽屉）**
  - 显示名称输入框（text）
  - 名称输入框（text，创建时可编辑，编辑时只读）
  - 描述输入框（text）
  - 基础指令编辑器（textarea，monospace 字体，约 20 行高度）
  - 模型配置区域：
    - 模型名称输入框（text，placeholder 显示当前默认值）
    - 温度滑块（range，0-2，步长 0.1）
    - API 密钥输入框（password，脱敏显示）
    - API 地址输入框（text）
  - 启用/禁用开关
  - "绑定技能"按钮（点击展开技能多选列表）
  - "保存"/"取消"按钮

- **技能绑定多选**
  - 调用 `GET /skills` 加载所有技能列表
  - 多选 checkbox 列表，显示技能名称和分类
  - 已选技能高亮
  - "确认绑定"按钮 → 调用 `POST /agents/{id}/bind-skills`

- **对话测试面板**
  - 消息输入框（text）
  - "发送"按钮
  - 回复展示区：Markdown 渲染的智能体回复
  - 加载状态指示（测试中...）
  - 智能体禁用时显示提示"智能体已禁用，请先启用"

#### 接口
- `GET /agents` — 列表加载
- `POST /agents` — 创建自定义智能体
- `PUT /agents/{id}` — 更新智能体配置
- `DELETE /agents/{id}` — 删除智能体
- `POST /agents/{id}/test` — 对话测试
- `POST /agents/{id}/bind-skills` — 绑定/解绑技能
- `GET /skills` — 加载技能列表（用于绑定多选）

#### 错误处理
- 创建/更新失败: 弹窗提示错误信息，保持编辑面板打开
- 删除内置智能体: 卡片不展示删除按钮
- 对话测试失败: 回复区显示错误信息（红色文字）
- 智能体已禁用时测试: 提示"智能体已禁用，请先启用后再测试"
- 网络错误: 统一 toast 提示"网络异常，请重试"

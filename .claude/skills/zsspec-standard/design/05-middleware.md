# 中间件设计

**版本**: v0.1

---

## 设计原则

| 原则 | 说明 |
|------|------|
| 统一入口 | 提供统一接口 |
| 配置外置 | 配置文件管理 |
| 异常处理 | 统一异常处理 |
| 单一职责 | 一个工具类一种中间件 |

## 目录结构

```
common/
├── config/    # 配置类
├── util/      # 工具类
└── constant/  # 常量
```

## Redis工具类

**类名**: CacheUtil

| 方法 | 说明 |
|------|------|
| set | 设置缓存 |
| get | 获取缓存 |
| delete | 删除缓存 |
| tryLock | 分布式锁 |

## MinIO工具类

**类名**: StorageUtil

| 方法 | 说明 |
|------|------|
| upload | 上传文件 |
| download | 下载文件 |
| delete | 删除文件 |

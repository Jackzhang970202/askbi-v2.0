## 业务流程

### 查询流程
接收参数 → 参数校验 → 构建条件 → 分页查询 → 返回结果

### 创建流程
接收数据 → 参数校验 → 业务校验 → 数据组装 → 持久化 → 返回ID

### 更新流程
接收ID和数据 → 校验 → 查原数据 → 业务校验 → 合并更新 → 返回结果

### 删除流程
接收ID → 查数据 → 业务校验 → 逻辑删除 → 返回结果
